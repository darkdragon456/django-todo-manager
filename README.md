# 🚀 Django Todo Manager

A full-stack **Django Todo Manager** that helps users organize tasks, track productivity, and stay motivated with a reward-based system.

## ✨ Features

* 🔐 User Registration & Login
* ➕ Create, Edit & Delete Tasks
* ✅ Mark Tasks as Completed
* 🏆 Reward-based productivity system
* 📊 Weekly productivity bar charts
* 📱 Responsive UI with Bootstrap
* 🔒 Authentication & Authorization
* 📧 Contact page with one-click email support
* 📍 Embedded Google Maps location

## 🚀 Run Locally

```bash
git clone https://github.com/darkdragon456/django-todo-manager.git
cd django-todo-manager
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
